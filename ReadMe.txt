Relazione: contiene la relazione nei formati file pdf e docx.
Moduli: contine classi python (contenute anche in BellomoClassificatore) separati in moduli.
BellomoClassificatore: contine il main contente la struttura del database e tutto ciò che riguarda all'allenamento della reta.
Backup: non utile al fine della consegna, contine una versione precedente del BellomoClassificatore.
Bellomo: contine il dataset.
Logs: contiene le informazioni per la costruzione dei grafici ottenute dall'allenamento della rete.
weight: contine i pesi per ciascun modello usato (non sono presenti i pesi di ogni epoca).

il video tutorial verrà consegnato insieme al link del git come allegato.
