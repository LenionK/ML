This README is related to the following scientific paper:
    
F. Ragusa, A. Furnari, S. Battiato, G. Signorello, G. M. Farinella. EGO-CH: Dataset and Fundamental Tasks for Visitors Behavioral Understanding using Egocentric Vision.
submitted to Pattern Recognition Letters

More information is available at our web page: http://iplab.dmi.unict.it/EGO-CH

For this cultural site you will find:

	1) Palazzo Bellomo:
		- Training folder which contains:
				-	images for all the 22 environments inside the x.y folders (where x indicates the environment)
				-	training/validation .txt which contain the labels for each frame

				
		- Test folder which contains: 
				-	10 folders representing the 10 test videos which contain the frames
				-	10 gt*.txt files which contain the labels for each test frame	
			

See the paper to more details.
